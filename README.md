#  Python Package Checker

A beginner-friendly Python project that uses the PyPI API to fetch information about Python packages.

This project demonstrates API integration, JSON handling, searching, and error handling using Python.



##  Features

- View details of any Python package from PyPI.
- Search for keywords in a package summary.
- Compare the latest versions of two packages.
- Handle invalid package names and internet errors.
- Parse JSON data into Python dictionaries.

---

## 🛠️ Technologies Used

- PyPI API
- JSON Handling
